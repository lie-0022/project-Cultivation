# 00. 게임 개요 (Overview)

## 1. Overview

3D 농장 시뮬레이션 + 가챠 + 크리처 컬렉션 + 교배 게임의 MVP. 플레이어가 3D 공간을 직접 돌아다니며 농장에서 작물을 재배하고, 그 작물을 크리처로 변환해 사육장에서 키우며, 교배를 통해 새로운 종을 발견하는 컬렉션 중심 게임.

- **장르**: 농장 / 가챠 / 크리처 컬렉션 / 교배 실험
- **플랫폼**: PC 다운로드형
- **엔진**: Unity (C#)
- **목표 플레이 길이**: 수백 시간 (장기 컬렉션 게임)
- **MVP 범위**: 핵심 게임 루프 1회전 + 3D 공간 이동/상호작용

## 2. Player Fantasy

"내가 만든 희귀종을 컬렉션하고 싶다."

- 식물을 길러 생명체로 만드는 신비로운 변환의 즐거움
- 두 종을 교배해 새로운 종을 발견하는 실험의 쾌감
- 농장을 직접 걸어 다니며 가꾸는 물리적 존재감
- 골드를 모아 더 많은 가챠/확장에 투자하는 점진적 성장

## 3. Detailed Rules

### 3.1 코어 루프

```
[가챠]
  → [씨앗 인벤토리]
  → [밭에 심기 (E키 상호작용)]
  → [작물 수확]
  → [크리처화]
  → [사육장 추가]
  → [교배] ←→ [크리처 컬렉션]
  → [판매 → 골드]
  → (가챠로 순환)
```

### 3.2 공간 구조

- **하나의 넓은 씬**에 모든 시설 배치
- 플레이어는 3D 캐릭터로 자유 이동 (WASD + Space, 데모 코드 제거 후 본 게임용으로 새로 구현)
- 시설물 종류:
  - **밭 영역**: 확장형 플롯들이 배치된 농장 구역
  - **사육장**: 크리처 슬롯들이 배치된 구역
  - **가챠 시설**: 씨앗 뽑기 NPC/건물
  - **상점/판매소**: 작물/크리처 판매 NPC/건물
  - **교배 연구소**: 크리처 교배 NPC/건물 (선택적, 사육장 내 통합도 가능)

### 3.3 상호작용 방식

- 플레이어가 시설물에 일정 거리 접근 → "E키로 상호작용" 프롬프트 표시
- E키 입력 → 해당 시설물의 UI 패널 열림 (가챠 UI, 사육장 UI 등)
- 스타듀밸리 방식

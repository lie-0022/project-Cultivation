# 결정 사항 요약 (Decisions)

기획 합의 후 확정된 결정 사항들. Claude Code 작업 시 이 문서가 진실 공급원.

---

## 게임 규칙 결정

| # | 항목 | 결정 |
|---|---|---|
| 1 | 가챠 묶음 | **1회만** (MVP는 단순) |
| 2 | 확장 비용 공식 currentCount 의미 | **직전 칸 수** (예: 2칸 보유 시 다음 비용 = 100*(2^1.5)) |
| 3 | 확장 최대치 | **무제한** (MVP), 추후 상한 도입 |
| 4 | 교배 후 부모 처리 | **잔존** (Busy 해제, 계속 사용 가능) |
| 5 | 동시 교배 슬롯 수 | **무제한** (사육장 모든 쌍 동시 교배 가능) |
| 6 | 교배 취소 가능 | **가능** (골드 전액 환불) |
| 7 | 작물 직접 판매 | **가능** (CropData.sellPrice 활용) |
| 8 | 같은 종 교배 시 우선순위 | **레시피 우선** (예: 토마토닭+토마토닭 → 케첩닭) |
| 9 | 게임 종료 시 진행 중 작업 | **알림 표시 후 종료** (다음 게임 시 모두 초기화, 저장 없음) |
| 10 | 판매 수량 | **1마리/1개씩만** (MVP) |

## MVP 비목표 (제외 항목)

다음 항목은 MVP 범위 외, 추후 추가:

- 형질 유전 (색상/크기/희귀도 변이)
- 사육장 환경 시스템 (온도/습도 등 종별 보너스)
- 도감 UI
- 시즌/이벤트
- 가챠 천장 시스템
- 가챠 10연차
- 근친 교배 패널티
- 저장/불러오기 (메모리 상에서만 동작)
- 정식 UI/UX (MVP는 임시 Button + Text)
- 사운드/이펙트
- 다중 판매 (종류별 일괄, 다중 선택)
- 확장 칸 상한
- 변환 비용/시간 (현재는 무료/즉시)
- NPC 대화/스토리
